# Profile Card Project

A clean and responsive profile card built with pure HTML and CSS. This project showcases a modern digital business card design with smooth hover effects and social media links.

## Features:
- ✅ Fully responsive design
- ✅ Smooth hover animations
- ✅ Social media icons
- ✅ Clean and modern UI
- ✅ Pure HTML & CSS (no JavaScript)

## Technologies Used:
- HTML5
- CSS3
- Flexbox/Grid Layout
